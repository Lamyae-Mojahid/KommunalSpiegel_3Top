# KommunalSpiegel – KS-Design integriert

Diese Version kombiniert:
- die funktionale KS-LiveGPT-FINAL3-App
- das gewünschte KS-Design aus `KS.zip`
- 33-Kommunen-Fallbackdaten
- Datenwerkstatt mit Agentenlauf
- IGEK/ISEK-Mapping, Maßnahmen, Förderung, Berichte und Quellen

## Start

```bash
cd ~/Downloads/KS-LiveGPT-FINAL3_KS_DESIGN_FINAL
npm install
npm run dev
```

Dann im Browser öffnen:

```text
http://localhost:3000
```

Wenn Port 3000 belegt ist, zeigt Next.js im Terminal automatisch einen anderen Port an.
